# タスク完了時のチェックリスト

## 必須実行コマンド

### 1. テスト実行
```bash
go test ./...                 # 全テストが通ることを確認
go test -v ./...             # 詳細出力でテスト結果確認
```

### 2. コード品質チェック
```bash
go fmt ./...                 # コードフォーマットの適用
go vet ./...                 # 静的解析でのエラー検出
```

### 3. モジュール整合性
```bash
go mod tidy                  # 不要な依存関係の削除
go mod verify                # モジュール依存関係の検証
```

### 4. ビルド確認
```bash
go build -o bin/app .        # ビルドエラーがないことを確認
```

## 任意実行項目

### アプリケーション動作確認
```bash
go run main.go               # HTTP サーバーが正常起動することを確認
```

### 特定テスト実行
```bash
go test -run TestSpecificName  # 修正した機能のテストを個別実行
```

## 注意事項
- **コミット前**: 必ず上記の必須コマンドを全て実行
- **テスト失敗時**: 修正してから次の作業へ進む
- **フォーマット**: `go fmt` は自動的にコードを整形
- **静的解析**: `go vet` のワーニングは解消する
